Optical Character Recognition Project (R)

This was a fun little project! Our office had about 800 vehicle titles that were not perfectly organized and we needed to make sure we had a title for each vehicle we owned. We also regularly receive stacks of 100+ titles that need to be filed and tracked. I wrote this short script using Tesseract 4 and some R code to extract VIN numbers and registered owner information Washington State vehicle titles to help with this process. This approach achieved 98%+ accuracy on VIN number extraction and 100% accuracy on registered owner information (higher then human level accuracy).
